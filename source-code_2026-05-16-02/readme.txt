FM TOWNS用のHIGH C V1.7のディレクトリのhc386を
まるごとPC9821エミュレータのNekoProject21/wにコピーして
PC9821上でコンパイルしてみました


FM TOWNSにインストールされたHigh CのフォルダをPC9821に持っていくことで
386以上のPC9801やPC9821でもFM TOWNS用の機種依存ライブラリを使わなければHigh Cが使えます



PC9801用のEXE386が必要になります

DOSエクステンダ EXE386 for PC-98
https://www.vector.co.jp/soft/maker/kmc/se002622.html

exe386は適当なディレクトリコピー後、LHAで解凍してPATHを通しておきます

FM TOWNSにインストールしてあるHigh Cのディレクトリhc386をそのままコピーしてきます

\hc386\binと\hc386\utilsにPATHを通す必要があります

\hc386\bin\hc386.cnfを編集して移動先の環境に合わせる必要があります

16行目と21行目を変更する必要があります

run386.exeはexe386.exeに変更してください
-nocrtがある場合は削除してください

環境変数IPATHにはヘッダーファイルのPATHを設定します
set IPATH=a:\hc386\inc



サンプルプログラムは
PC9821の640x400 256色モードでメキシカンハットを描くプログラムです
PC9821では特定の操作をすることにより、0x00F00000のアドレスにVRAMが現れる仕様になってます
VRAMは0x00F00000から512KBでリニアアクセスが可能になります
今回のサンプルプログラムはこの機能を利用して描画してます

サンプルプログラムではVRAMを新しく取得したセグメントにマッピングしてVRAM操作をしています
新しいセグメントの取得するAPI、ハードウェアをセグメントにマッピングするAPI
セグメントの領域とセグメントを開放するAPIの3つのEXE386のAPIを呼び出す関数はアセンブラで記述してます

詳しくはソースコードを見てみてください
