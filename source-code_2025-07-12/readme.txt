また、別売のJPEGサポートライブラリが別売されていました


1994年に発売されたHigh C Compilerマルチメディアキット
またはHigh C Compilerマルチメディア開発キットには
これらのライブラリ一式が付属してます


JPEGサポートライブラリを使って簡単なJPEG画像読み込みプログラムを作ってみました
画面モードは10の320x240、32768色モードを使ってます


コンパイル方法
hc386 -v -c TJPGREAD.C
DLRC -OBJ .\TJPGREAD.RCO TJPGREAD.RDF
386link @TJPGREAD

注意:TJPGREAD.LNKの中のライブラリへのパスは各自環境に合わせて変更してください


実行方法

run386 TJPGREAD image01.jpg
