各デモプログラムの冒頭に説明とコンパイル方法が記載してあります。

i2ctest_bme280.cはI2Cで気温、湿度、気圧センサーのBME280を制御してます。
i2ctest_ssd1306.ccはI2CでOLEDのSSD1306を制御してます。
i2ctest_ssd1306_bme280.ccはBME280で読み取ったデータをSSD1306に表示するプログラムです。
spitest_ili9341_double_buffer.cはLCDのILI9341に画像を表示するプログラムです。
spitest_ili9341_double_buffer_draw_char.ccはILI9341に文字を表示するデモプログラムです。
spitest_ili9341_mandelbrot_set03.cはILI9341にマンデルブロ集合を表示するプログラムです。
spitest_ili9486_double_buffer.cはLCDのILI9486に画像を表示するプログラムです。
spitest_ili9486_double_buffer_draw_char.ccはILI9486に文字を表示するデモプログラムです。
spitest_ili9486_mandelbrot_set03.cはILI9486にマンデルブロ集合を表示するプログラムです。

yamaから始まるファイル名はライブラリ的な部分を分離したものです。
