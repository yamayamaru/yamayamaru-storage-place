電脳伝説さんのところのGrant's BASICを改変して
https://github.com/vintagechips/aki80basic
Super AKI-80で秋月電子で販売されているAKI-80用のROMを使った場合に
RAM上でGrants'sを動くようにしてみました
以前、私がBabbage2号で使ったソースコードに少し手を加えただけです
MicrosoftのアセンブラのM80用にしてM80でアセンブルしました



プログラムの実行方法
秋月電子でAKI-80用のROMをSuper AKI-80に挿して
パソコンにつないだUSB-SerialのアダプタにAKI-80のシリアルポートをつなぎます
パソコンではTeratermを使ってシリアルスピードを9600bpsにして待機します

AKI-80に電源を入れてAKI-80をリセットします
@と表示されたらROMのモニタが起動した状態です
Teratermのメニューの設定→端末の画面で
改行コードの送信をLFにしてローカルエコーにチェックを付けます

この状態で何かキーを押すと
RS
と表示されます

LHと入力してEnterキーを押します
$と表示されるのでTeratermのメニューのファイル→ファイル送信でBASICのインテルHEXファイルを送信します
送信が終わると@が表示されます
一度Enterキーを押します

GO8000と入力してEnterキーを押すとBASICが起動します


Basicが起動したらTeratermのメニューの設定→端末の画面の改行コードの送信をCRにして
ローカルエコーのチェックを外します
