OpenWatcom C/C++ Ver2.0でEXE386用のPC9821の256色パックドピクセル用のプログラムを作成してみました

PC9821の256色パックドピクセルのモードではVRAMをアドレス0x00F00000に出現させてVRAMにリニアにアクセスすることができます
PC9801用のEXE386が必要になります

DOSエクステンダ EXE386 for PC-98
https://www.vector.co.jp/soft/maker/kmc/se002622.html

exe386は適当なディレクトリコピー後、LHAで解凍してPATHを通しておきます

サンプルプログラムは
PC9821の640x400 256色モードでメキシカンハットを描くプログラムです
PC9821では特定の操作をすることにより、0x00F00000のアドレスにVRAMが現れる仕様になってます
VRAMは0x00F00000から512KBでリニアアクセスが可能になります
今回のサンプルプログラムはこの機能を利用して描画してます

サンプルプログラムではVRAMを新しく取得したセグメントにマッピングしてVRAM操作をしています
新しいセグメントの取得するAPI、ハードウェアをセグメントにマッピングするAPI
セグメントの領域とセグメントを開放するAPIの3つのEXE386のAPIを呼び出す関数はアセンブラで記述してます
