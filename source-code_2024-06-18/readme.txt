Turbo C(Borland C)によるPC9801用のマンデルブロー集合のプログラムです


コンパイル方法は
tcc -O2 mandelg2.c graphics.lib


BGIも付けておきます
BGIファイルは実行するプログラムと同じディレクトリに置いてください

DOSVGA       DOSVGA_driver
PC98         PC98_driver
PC98HI       PC98HI_driver

これらのプログラムの使用に当たってはご自分の責任において使用してください
