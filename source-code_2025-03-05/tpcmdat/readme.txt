Oh! TOWNS 1993年4月号 Cの散歩道の第6回 スプライトライブラリの基礎の
サンプルprogramを少しアレンジして200個のスプライト表示と
サウンドの再生するプログラムを作ってみました

スプライトを使う場合の参考になると思います
また、大容量のサウンドデータの再生のサンプルにもなると思います

tpcmdat.batを実行するとコンパイルできますが
tpcmdat.batやtpcmdat.lnkのリンカへのパスやライブラリへのパスは
ご自分の環境に合わせて変更する必要があります
