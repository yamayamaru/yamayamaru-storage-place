GS ResetやXG ONを送ることができる簡単なアプリです
また、midiのデバイス番号を調べることができます
リストボックスをクリックすると選択可能なMIDIデバイスのリストが表示されるので
GS ResetやXG ONを送りたいMIDIデバイスを選択します

GS Resetボタンを押すとGS Resetが選択されたMIDIデバイスに出力されます
XG ONボタンを押すとXG ONが選択されたMIDIデバイスに出力されます


コンパイルにはVisual Studio Communityが必要です

コンパイル方法
x64 Native Tools Command Prompt VS 20xxを立ち上げてソースのあるフィルダに移動して

cl win_midiout_reset.cpp

とするだけです
