Oh! FM 1990年7月号のp88からp96に若松登志樹さんのπの計算プログラムが掲載されています

このπの計算プログラムはオールアセンブラで記述されているのですが
無理矢理C言語に移植してみました
アセンブラのプログラムよりも1.4倍ほど遅くなってます

OpenWatcom C Ver2.0用、High C V1.7用、64bitCPUのGCC用を用意してみました


High C用のコンパイルですがpimil.lnkを開いて-libの部分をご自分の環境に合わせてパスを変更してください
また、アセンブラで記述されているものはMASMやTASMでアセンブルを試しました

64bitCPUのGCC用はアセンブラのコードは一切なく、すべてC言語で記述されてます
UbuntuのRaspberryPi4、UbuntuのVisionFive2でコンパイル、実行を確認してます
